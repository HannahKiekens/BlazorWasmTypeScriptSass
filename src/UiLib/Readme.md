# Update the filles in wwwroot
``` bash
npm install

# compile ts/js files
npm run build-scripts
# compile sass files
npm run build-css
```
* TS entry file: Scripts/UiLib.ts
* SCSS entry file: Styles/UiLib.scss

### Windows
Ugly workaround for npm on windows:
``` bash
npm install -g webpack-cli webpack
```
